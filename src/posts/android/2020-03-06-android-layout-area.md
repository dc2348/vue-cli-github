# Android 하이브리드 앱의 네이티브 영역과 웹뷰 영역 구분하기


### 개발자 도구 실행
1. **설정** 앱 실행
2. **개발자 옵션** 메뉴 선택
3. **레이아웃 범위 표시** on으로 변경
4. 분석하고 싶은 앱 실행
