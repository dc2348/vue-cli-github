# ESLint에서 특정 파일 형식 제외하기

- 프로젝트의 루트 디렉토리에 `.eslintignore` 파일을 작성하여 특정 파일 및 디렉토리를 무시하도록 ESLint에 지시 할 수 있다. 
- 예를 들어, 다음은 모든 JavaScript 파일을 생략한다.
    ```
    **/*.js
    ```
<br>
