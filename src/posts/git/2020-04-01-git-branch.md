# Git 브랜치

## branch 생성
```bash
git branch branchname
```

## 현재 작업 중인 branch 확인
```bash
git log --oneline --decorate
```
- `HEAD ->`가 가리키고 있는 branch를 확인하면된다.


## branch 이동하기
```bash
git checkout branchname
```
